# Introdução ao Node

## Introdução a criação de API Restfull com nodejs

Nesse módulo aprenderemos conceitos básicos em uma aplicação com Node, como por
exemplo:

- Métodos http (GET, POST, PUT e DELETE)
- Código de resposta
  - 1xx : Information
  - 2xx : Sucess
  - 3xx : Redirection
  - 4xx : Client Error
  - 5xx : Server Error
- Query e Route Params
- Middlewares
